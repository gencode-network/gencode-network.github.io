Join the effort to build a community-scale genomics platform

GenCoDE is a collaborative platform built on shared data and insight. We invite researchers to contribute genomic datasets that can advance community-level genomic science.

By contributing to GenCoDE, you are helping build a structured, interoperable resource designed to uncover eco-evolutionary patterns that are not visible within single-species datasets alone.
